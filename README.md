# The Legend of Textica

Textica is a text-based RPG that can be played using the C# console. The purposes of this project is to demonstrate both basic and object-oriented programming principles in C#. The program is functional, but there may be bugs from running the program.

If you're using Visual Studio Code, you can clone or download the repo as a .zip file and use `dotnet run` in the terminal. If using Visual Studio, do the same as before and open the Textica.sln solution file; you can then click on the green arrow to run the program.
